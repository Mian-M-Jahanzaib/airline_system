# UOG Airline Management System

A modern, responsive web application designed for an airline management system. This project demonstrates a frontend interface built for booking flights, viewing airline information, and providing customer support.

## Key Features

- **Responsive Design:** A fully responsive layout that adapts seamlessly to desktop, tablet, and mobile screens.
- **Dynamic Navigation:** An interactive Navbar with smooth transitions and dropdown menus.
- **Hero Section:** A minimalist, thematic landing area that establishes the brand identity immediately.
- **About Us & Fleet Info:** Dedicated sections providing detailed information about the airline's mission and fleet.
- **Contact Support:** A functional and user-friendly contact form layout for customer inquiries.
- **Modern Styling:** Utilizes Tailwind CSS for a clean, professional aesthetic with custom animations.

## Technologies Used

- **Frontend Framework:** React.js
- **Styling:** Tailwind CSS
- **Routing:** React Router DOM
- **Icons:** React Icons
